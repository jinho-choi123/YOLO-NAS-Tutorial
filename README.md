## YOLO-NAS tutorial

### Getting Started
#### Prerequisites
1. install [uv-python](https://github.com/astral-sh/uv)
2. Use python version 3.10.11
```
$ python --version
Python 3.10.11
```
3. setup virtual environment using uv-python
```
$ uv venv
```

#### Run the Code
1. Install required packages
```
$ uv pip install -r requirements.txt
```
2. Run the code
```
$ python main.py
```
